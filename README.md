# 📌 프로젝트 소개

서울에는 **25개 자치구**가 있습니다.

단순히 **따릉이를 많이 이용한 지역**을 찾는 것은 어렵지 않습니다.

하지만 이 프로젝트는 한 걸음 더 나아가,

> **"어느 지역에서 따릉이가 생활 속 교통수단으로 가장 잘 자리 잡았는가?"**

를 데이터로 분석합니다.

이를 위해 자체 지표인 **Bike Life Index(따릉이 생활지수)** 를 설계하여
서울시 자치구를 비교하고 인사이트를 제공합니다.

# 📊 Bike Life Index란?

> **따릉이가 해당 지역에서 생활 교통수단으로 얼마나 자리 잡았는지를 나타내는 종합지표입니다.**

생활지수는 아래 네 가지 요소를 종합하여 계산합니다.

| 구성 요소 | 비중 |
|-----------|------:|
| 🚲 이용 활성도 | **40%** |
| 📅 연중 지속성 | **30%** |
| 📈 성장성 | **20%** |
| 🌤 계절 안정성 | **10%** |

---

# 🏅 Bike Life Grade

| 점수 | 등급 | 의미 |
|------|------|------------------------|
| 90 ~ 100 | 🟢 A+ | 생활 정착형 |
| 80 ~ 89 | 🟢 A | 생활 활성형 |
| 70 ~ 79 | 🟡 B | 성장 생활형 |
| 60 ~ 69 | 🟠 C | 생활 잠재형 |
| ~59 | 🔴 D | 생활 기반 형성형 |
