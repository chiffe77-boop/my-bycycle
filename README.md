# my-bycycle
# 📊 연령별 인구현황 대시보드

행정안전부 **연령별 인구현황(주민등록 인구통계)** CSV 데이터를 활용해, 원하는 지역(시/도 ~ 읍/면/동 단위)의 인구 구조를 한눈에 살펴보고, 전국에서 인구 구조가 가장 비슷한 지역 Top 5까지 찾아주는 Streamlit 웹앱입니다.

Streamlit Community Cloud에 바로 배포할 수 있도록 구성되어 있습니다.

---

## ✨ 주요 기능

- **지역 검색 + 다중 선택**: 사이드바에서 지역명을 일부만 입력해도 목록이 좁혀지고, 드롭다운(멀티셀렉트)에서 여러 지역을 동시에 골라 비교할 수 있습니다. (시/도, 시/군/구, 읍/면/동 모두 포함)
- **연령별 인구 구조 꺾은선 그래프**: 선택한 지역들의 0세 ~ 100세 이상 인구수를 Plotly 꺾은선 그래프로 시각화합니다. 계/남/여 기준을 선택할 수 있습니다.
- **선택 지역 요약 지표**: 선택한 각 지역의 총인구수를 카드 형태로 보여줍니다.
- **전국 인구구조 유사 지역 Top 5**: 연령대별 인구 "비율(구조)"을 코사인 유사도로 비교하여, 선택한 기준 지역과 인구 구조가 가장 비슷한 전국 지역 Top 5를 찾아 그래프와 표로 보여줍니다. (인구 규모가 아닌 구조의 "모양"을 기준으로 비교하기 때문에 규모가 다른 지역끼리도 공정하게 비교됩니다)
- **원본 데이터 테이블 보기**: 필요 시 선택 지역의 원본 데이터를 표로 확인할 수 있습니다.

```

├── 202606_202606_연령별인구현황_월간.csv     # 데이터 파일 (app.py와 같은 폴더)
└── README.md
```

> `app.py`는 실행 시 코드 파일과 **같은 폴더**에 있는 CSV를 자동으로 찾아 로드합니다. 파일이 없으면 화면에서 직접 업로드할 수도 있습니다.

## 🗂 데이터 출처

- [행정안전부 주민등록 인구통계](https://jumin.mois.go.kr/) 에서 내려받은 **연령별 인구현황(월간)** CSV
- 인코딩: `CP949(EUC-KR)`, 숫자는 천 단위 콤마(`,`) 포함
- 행: 전국 시/도 · 시/군/구 · 읍/면/동 단위 행정구역 (3,918개)
- 열: 지역별 계/남/여 기준 0세 ~ 100세 이상 연령별 인구수 (약 310개 컬럼)

1. 이 저장소(`app.py`, `requirements.txt`, CSV 파일 포함)를 GitHub에 push
2. [share.streamlit.io](https://share.streamlit.io) 접속 → **New app**

4. 배포 완료 후 발급되는 URL로 접속

## 📐 유사 지역 계산 방식

1. 각 지역의 연령별 인구수를 해당 지역 총인구수로 나눠 **연령대별 비율(구조) 벡터**로 정규화합니다.
2. 기준 지역의 벡터와 전국 모든 지역(시/도 · 시/군/구 · 읍/면/동)의 벡터 간 **코사인 유사도**를 계산합니다.
3. 유사도가 높은 순서대로 Top 5 지역을 추출해 그래프와 표로 제공합니다.

이 방식은 인구 규모(총인구수)가 아니라 **인구 구조의 모양**을 비교하기 때문에, 대도시와 소규모 읍/면/동이라도 연령 분포 패턴이 비슷하면 유사 지역으로 매칭됩니다.

## 📝 참고 사항

- CSV 파일명이 바뀌면 `app.py` 상단의 `DEFAULT_FILENAME` 값을 함께 수정해야 합니다.
- 데이터 갱신 시(예: 다음 달 CSV로 교체) 컬럼명의 연도·월 표기(`2026년06월_...`)가 다르면 연령 컬럼 추출 로직(`_{gender}_` 마커 기반)은 그대로 동작하지만, 총인구수 컬럼명 매칭(`_총인구수`로 끝나는 컬럼) 로직도 함께 확인하는 것을 권장합니다.
