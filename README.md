# 🚲 서울 따릉이 생활지수 (Seoul Ddareungi Life Index)

> **서울 열린데이터광장의 공공데이터를 활용하여 자치구별 따릉이 이용 패턴을 분석하고, 생활지수(Bike Life Index)를 통해 서울의 자전거 문화를 시각화한 데이터 분석 웹앱입니다.**

---

## 📌 Preview

**서울에서 따릉이가 가장 생활 속 교통수단으로 자리 잡은 곳은 어디일까요?**

이 프로젝트는 단순한 이용량 순위를 보여주는 것이 아니라,

- 🚲 얼마나 많이 이용되는지
- 📈 얼마나 꾸준히 이용되는지
- 🌱 얼마나 성장하고 있는지
- 📅 계절 영향을 얼마나 적게 받는지

를 종합하여 **Bike Life Index(생활지수)** 를 계산합니다.

---

# ✨ Features

✅ 서울 전체 대시보드

✅ 자치구별 따릉이 생활지수

✅ 월별 이용패턴 분석

✅ 나와 비슷한 자치구 추천

✅ 숨은 강자 찾기

✅ AI 자동 인사이트

✅ 성장 시뮬레이션

---

# 📊 Bike Life Index

생활지수는 단순 이용량이 아닙니다.

> **"따릉이가 해당 지역에서 생활 교통수단으로 얼마나 자리 잡았는가"**

를 의미하는 종합지표입니다.

| 항목 | 비중 |
|------|------:|
| 🚲 이용 활성도 | 40% |
| 📅 연중 지속성 | 30% |
| 📈 성장성 | 20% |
| 🌤 계절 안정성 | 10% |

---

# 🏆 Bike Life Grade

| 점수 | 등급 | 의미 |
|------|------|--------------------|
| 90~100 | 🚲🚲🚲🚲🚲 | 생활 정착형 |
| 80~89 | 🚲🚲🚲🚲 | 생활 활성형 |
| 70~79 | 🚲🚲🚲 | 성장 생활형 |
| 60~69 | 🚲🚲 | 생활 잠재형 |
| 60 미만 | 🚲 | 생활 기반 형성형 |
