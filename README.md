# 🚲 서울 따릉이 생활지수 (Seoul Ddareungi Intelligence)
### Seoul Ddareungi Life Index

> **서울시 공공데이터를 활용하여 자치구별 따릉이 이용 패턴을 분석하고,
AI 기반 인사이트를 제공하는 데이터 분석 웹앱**

---

## 📌 프로젝트 소개

서울 열린데이터광장의 **공공자전거 자치구별 대여건수(2021)** 데이터를 활용하여

단순한 이용량 조회를 넘어

- 어느 자치구가 가장 활발하게 이용되는지
- 어떤 지역이 꾸준히 성장하고 있는지
- 나와 가장 비슷한 이용패턴을 가진 지역은 어디인지
- 생활지수(Bike Life Index)는 얼마나 되는지

를 한눈에 확인할 수 있는 **데이터 인사이트 서비스**입니다.

---

## 🎯 주요 기능

### 🚲 서울 따릉이 생활지수

- 생활지수 자동 계산
- 자치구 순위
- 성장성 분석
- 지속성 분석

---

### 📈 월별 이용패턴 분석

- 월별 대여건수
- 계절별 변화
- Peak Month 분석
- 연중 이용패턴

---

### 🔍 나와 비슷한 자치구

선택한 자치구와

- 이용규모
- 성장률
- 계절성
- 지속성

을 비교하여 가장 유사한 자치구를 추천합니다.

---

### 🧠 AI 데이터 인사이트

선택한 자치구를 분석하여

- 특징
- 성장성
- 계절성
- 강점
- 개선 포인트

를 자연어로 제공합니다.

---

### ⭐ 숨은 강자 찾기

단순 순위가 아닌

**예상 이용량 대비 실제 이용량**

을 분석하여

잠재력이 높은 자치구를 찾아냅니다.

---

### 📊 성장 시뮬레이션

슬라이더를 이용하여

대여건수가 증가하거나 감소했을 때

- 생활지수
- 순위
- 서울 전체 영향

을 확인할 수 있습니다.
