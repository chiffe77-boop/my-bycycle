🚲 서울 따릉이 생활지수 (Seoul Ddareungi Intelligence)
따릉이는 우리 동네의 일상이 되었을까?
서울시 공공자전거 데이터를 활용해 따릉이가 각 자치구에서 얼마나 활발하고, 꾸준하며, 성장하고, 계절 변화에도 안정적으로 이용되는지 분석하는 Streamlit 데이터 웹앱입니다.

---
프로젝트 소개
서울 따릉이 생활지수는 단순히 대여건수가 많은 자치구를 보여주는 순위가 아닙니다.
> 따릉이가 한 지역에서 일상적인 이동수단으로 얼마나 자리 잡았는지를 네 가지 관점으로 종합한 상대평가 지수입니다.
이 앱은 서울 25개 자치구의 월별·연간 따릉이 대여 데이터를 분석하여 다음 질문에 답합니다.
어느 자치구에서 따릉이가 가장 활발하게 이용되는가?
특정 계절에만 집중되지 않고 일 년 내내 꾸준히 이용되는가?
전년보다 이용이 성장하고 있는가?
나와 비슷한 이용 패턴을 가진 자치구는 어디인가?
단순 이용량 순위에서는 발견하기 어려운 숨은 강자는 어디인가?
---
서울 따릉이 생활지수의 의미
생활지수는 아래 네 요소를 0~100점의 상대점수로 변환한 뒤 가중 합산합니다.
구성요소	가중치	의미	계산 관점
이용 활성도	40%	얼마나 많이 이용되는가	2021년 연간 대여건수
연중 지속성	30%	비수기에도 이용이 유지되는가	최저 이용월 ÷ 월평균 이용량
성장성	20%	생활 교통수단으로 확장되고 있는가	2020년 대비 2021년 증가율
계절 안정성	10%	계절에 따른 이용 변동이 작은가	월별 변동계수의 역방향 점수
해석 원칙
이용 활성도가 높으면 실제 이용 규모가 큰 지역입니다.
연중 지속성이 높으면 가장 한산한 달에도 일정 수준의 이용이 유지됩니다.
성장성이 높으면 전년보다 이용 기반이 빠르게 확대되고 있습니다.
계절 안정성이 높으면 특정 계절이나 날씨에 덜 의존하는 생활형 패턴에 가깝습니다.
따라서 생활지수가 높다는 것은 단순히 많이 탔다는 뜻이 아니라, 많이 이용되고 비수기에도 유지되며 성장성과 안정성을 함께 갖춘 지역이라는 의미입니다.

생활지수 등급
점수	등급	해석
90점 이상	A+	생활 정착형
80~89.9점	A	생활 활성형
70~79.9점	B	성장 생활형
60~69.9점	C	생활 잠재형
60점 미만	D	생활 기반 형성형
등급과 점수는 서울 25개 자치구 안에서 비교한 탐색적 상대평가입니다.

주요 기능
1. 서울 한눈에
서울 전체 대여건수와 최고 이용월
자치구별 연간 대여건수 지도
이용량 상위 자치구 순위
서울 전체 월별 이용 흐름
2. 생활지수 분석
자치구별 생활지수와 순위
A+~D 등급 및 생활유형
이용 활성도·연중 지속성·성장성·계절 안정성 비교
선택 자치구의 레이더 차트
3. 월별·계절별 이용 패턴
선택 자치구와 서울 평균 비교
최고·최저 이용월 탐색
봄·여름·가을·겨울 이용 비중 분석
4. 비슷한 자치구 추천
월별 이용 비중과 규모, 계절성, 성장성을 표준화해 선택 자치구와 가장 비슷한 지역을 추천합니다.
5. 이용 유형 분류
K-means 군집분석으로 자치구를 비슷한 이용 패턴끼리 분류합니다.
연중 생활형
계절 집중형
성장 가속형
대규모 수요형 등
6. 숨은 강자 탐색
연중 지속성·계절 안정성·성장성으로 기대 이용량을 추정한 뒤 실제 이용량이 기대보다 높은 자치구를 찾습니다.
7. 성장 시뮬레이션
선택 자치구의 연간 대여건수가 변할 때 다음 결과를 즉시 계산합니다.
예상 생활지수
예상 순위
서울 전체 이용량 변화
8. 자동 인사이트
선택 자치구의 주요 특징을 문장으로 요약하고 데이터 기반 실행 제안을 제공합니다.

프로젝트 파일
GitHub 저장소 최상단에 아래 파일을 그대로 업로드하세요.
```text
.
├── main.py
├── requirements.txt
├── README.md
└── 서울시 공공자전거 자치구별 대여건수(2021년).xlsx
```
> 데이터 파일명은 코드와 정확히 일치해야 하므로 변경하지 마세요.
코드는 아래 파일명을 사용합니다.
```python
DATA_FILE = Path(__file__).parent / "서울시 공공자전거 자치구별 대여건수(2021년).xlsx"
```

로컬 실행 방법
1. 패키지 설치
```bash
pip install -r requirements.txt
```
2. Streamlit 실행
```bash
streamlit run main.py
```

Streamlit Community Cloud 배포
GitHub에 새 저장소를 생성합니다.
`main.py`, `requirements.txt`, `README.md`, 데이터 파일을 저장소 최상단에 업로드합니다.
Streamlit Community Cloud에서 Create app을 선택합니다.
GitHub 저장소를 연결합니다.
Main file path에 `main.py`를 입력합니다.
Deploy를 누릅니다.
별도의 API Key나 Streamlit Secrets 설정은 필요하지 않습니다.
서울 지도 GeoJSON은 앱 실행 시 공개 저장소에서 불러옵니다. 지도 데이터를 가져오지 못할 경우 앱은 자동으로 순위 그래프로 대체합니다.

데이터와 분석의 한계
현재 생활지수는 제공된 대여건수 데이터만으로 계산합니다. 따라서 다음을 직접 측정하는 공식 정책지표는 아닙니다.
주민 1인당 이용량
대여소 1개당 이용 효율
자전거도로 접근성
출퇴근 목적과 레저 목적의 구분
실제 이용자 수와 반복 이용률
인구, 면적, 대여소 수, 자전거도로 길이, 지하철역 수 등의 데이터를 결합하면 생활지수를 더욱 정교한 도시 이동성 지표로 발전시킬 수 있습니다.
